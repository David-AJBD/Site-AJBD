# AJBD – Site statique

Ce bundle contient un site statique (HTML/CSS) prêt à être déposé tel quel sur n’importe quel hébergement (OVH, Infomaniak, GitHub Pages, Webador en page HTML, etc.).

## Fichiers
- `index.html` : page principale (one‑page)
- `styles.css` : styles avec variables de marque
- `assets/ajbd-*.jpg` : logos fournis (vert/rose/bleu/noir)

## Personnalisation
- Couleurs de marque : modifiez les variables CSS dans `styles.css`
- Logo header/footer : remplacez `assets/ajbd-vert.jpg` par la version souhaitée (rose/bleu/noir) et ajustez le `src` dans `index.html`.
- Sections : éditez le HTML pour contenus, cas clients et images.

## Déploiement
Copiez le dossier sur votre hébergement. Aucun build n’est requis.
